Soybeans as Second Commodity

Full end-to-end soybean support. The oracle scoring engine is now commodity-agnostic, with pluggable field definitions per commodity type. Adding a new commodity is a single config object.

  • Submission form: Cannabis/Soybeans toggle with commodity-specific fields (variety, protein, moisture, oil, USDA grade, storage, delivery terms)
  • Launchpad: Commodity filter chips, soybean-appropriate imagery and metrics
  • Exchange: SOY-USDC trading pair at $11.42 with commodity-aware volatility
  • Database: 12 new columns on launchpad_listings

Soybean Price Oracle

No on-chain soybean price feed exists anywhere. We built a custom oracle edge function that fetches from FRED/USDA and World Bank, computes a confidence-weighted composite price, detects anomalies, and updates both the oracle table and SOY-USDC ticker.

  • Schedule: pg_cron every 15 minutes, running live now
  • Sources: FRED/USDA (monthly), World Bank (daily), simulated fallback
  • Pyth outreach: Email drafted requesting CME ZS soybean futures feeds (they already have corn)

Milestone-Based Escrow

Funds are held in escrow and released at verified checkpoints, not all upfront. Four milestones per crop cycle.

MilestoneReleaseTrigger
Planting25%Verified by oracle/auditor
Growth25%Verified by oracle/auditor
Harvest35%Verified by oracle/auditor
Delivery15%Verified by oracle/auditor

Helius webhook automatically confirms deposits and releases when transactions land on Solana. Three new Supabase tables: escrow_accounts, escrow_deposits, escrow_milestones.
